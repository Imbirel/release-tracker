# Release Tracker

This is an app for tracking new releases in the field of films, games, and other things.


## About The Project



### Built With

- Vite (with Rolldown) v7.2.5
- React v19.2
- TS v5.9.3
- Tailwind CSS v4.1.17

## Browser support

You can configure browser support in the file `.browserslistrc`.
The project uses Tailwind CSS v4. He is designed for Safari 16.4+, Chrome 111+, and Firefox 128+. If you need to support older browsers, please downgrade to v3 until your browser support requirements change.


## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

1. npm

```sh
npm install npm@latest -g
```

2. Node.js v22.6.0 and up (with flag) or v23.6.0 and up

This is required because we use the `.ts` file extension for configuration `lint-staged.config.ts`.
You can read more about it [here](https://github.com/lint-staged/lint-staged?tab=readme-ov-file#typescript).


### Installation

1. Clone the repo

```sh
git clone https://github.com/Imbirel/release-tracker.git
```

2. Install NPM packages

```sh
npm install
```
